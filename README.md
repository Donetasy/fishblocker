# Auto Fisher Mod

A lightweight and efficient fishing automation mod that automatically reels in caught fish and recasts your fishing rod, allowing for continuous fishing with minimal user interaction.

> **Disclaimer:** This mod is intended for single-player or servers where automation is permitted. Always review and follow the rules of any server before using this mod.

---

## Features

- Automatically detects when a fish is caught
- Reels in the fishing rod automatically
- Recasts the fishing rod after each catch
- Lightweight and optimized for performance
- Configurable settings
- Simple toggle for enabling or disabling the mod

---

## Installation

1. Install the required mod loader.
2. Download the latest release from the Releases page.
3. Place the `.jar` file into your Minecraft `mods` folder.
4. Launch Minecraft.

---

## Usage

1. Join a world or server.
2. Equip a fishing rod.
3. Cast your line.
4. Enable the Auto Fisher.
5. The mod will automatically reel in fish and recast the rod.

---

## Configuration

The mod includes configurable options such as:

- WaitRecast
- WaitCatch
- StableTicks

Configuration values can be adjusted to suit your preferences.

## Building

Clone the repository and build using Gradle:

```bash

cd YourRepository
./gradlew build
```

The compiled mod will be located in:

```
build/libs/
```
